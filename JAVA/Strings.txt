Strings
-------
A string is a group of characters

names, products, email, address

Strings ---> char array  ---> every manipulation need to implemented by the users
 or developers

In Java string is a class type

class can have some predefined methods, by using them the strings can be processed
more  effectively and  efficiently.

numbers(without decimals)---> int, short, byte, long 
numbers with decimals--> float, double
true or false -- Boolean
group of chars (String) ----> String, StringBuffer, StringBuilder --= classes (Complex Structure) 


Strings can be handled in 3 ways:

1. String
2. StringBuffer
3. StringBuilder

if str1 comes later str2 in alpabetcical order ---> -ve value
if str1 == str2 then it returns zero value
